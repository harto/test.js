test.js by harto

A small, JUnit-like unit test framework for JavaScript.

test.js is a lightweight, JUnit-like testing framework for JavaScript. It can run within or without a web browser. It aims to provide the simplest possible API for unit test developers. test.js was partly inspired by jsUnity: http://code.google.com/p/jsunity/
